prime(x) 함수 전체 코드와 단계별 설명

Copy code

function prime(x) {

var i;

var count = 0;

if (x == 2)

return true;

for (i = 1; i <= Math.sqrt(x); i += 2) {

if (x % i == 0)

count += 1;

if (count >= 2)

return false;

}

return true;

}

이 함수는 x 가 소수인지 아닌지 true 또는 false 로 반환해 줍니다.

1. 숫자 2 는 특별 처리 (if (x == 2))

• 설명: 2 는 소수 중에서 유일한 짝수입니다. 따라서, 이 함수는 2 가 입력되면 더 이상

확인하지 않고 바로 true 를 반환하여 2 가 소수임을 알려줍니다.

• 목적: 2 를 처리하여 이후의 계산을 단순화하고 성능을 조금 더 빠르게 합니다.

2. 홀수만 확인 (for 루프에서 i += 2)

• 설명: for 루프는 i 를 1 부터 시작해 Math.sqrt(x) (즉, x 의 제곱근)까지만 확인합니다. 모든

짝수는 소수가 될 수 없으므로 짝수는 건너뛰고 i 를 2 씩 증가시켜 홀수만 검사합니다.

• 목적: 모든 짝수를 건너뛰어 루프를 빠르게 돌도록 하여 효율성을 높입니다.

3. 나누어 떨어지는지 확인 (if (x % i == 0))

• 설명: x % i == 0 은 x 가 i 로 나누어 떨어지는지 확인하는 조건입니다. 이 조건이 true 면 i 는

x 의 약수입니다.

• 목적: 약수의 개수를 확인하여 소수인지 여부를 판별합니다.

4. 소수가 아닌 경우 (count >= 2)

• 설명: count 는 x 의 약수의 개수를 셉니다. 만약 count 가 2 이상이 되면 x 는 더 이상 소수가

아니므로 false 를 반환합니다. 소수는 1 과 자기 자신만 약수로 가지므로, count 가 1 이하일

때만 소수일 수 있습니다.

• 목적: 약수의 개수를 통해 효율적으로 소수 여부를 빠르게 결정합니다.

5. 끝까지 조건에 걸리지 않으면 소수로 판별 (return true)

• 설명: 루프가 끝날 때까지 count 가 2 이상이 되지 않았다면, x 는 소수입니다. 따라서, return

true 로 소수임을 나타냅니다.

• 목적: 모든 검사를 마치고도 false 를 반환하지 않은 경우 소수라는 결론을 내립니다.

factor() 함수는 입력된 숫자를 소인수분해하여 결과를 계산하고, HTML 로 출력하는 기능을 합니다.

이 함수는 가장 작은 소수부터 시작하여 숫자를 계속 나누면서 소인수를 구하는 방식으로

작동합니다. 이제 코드의 주요 부분을 단계별로 설명하겠습니다.

전체 코드와 단계별 설명

javascript

Copy code

function factor() {

var x = document.getElementById("num").value, // 사용자가 입력한 값

i = 2, // 가장 작은 소수로 시작

exp = 0; // 소수의 지수 초기화

var str = x + " = "; // 결과 문자열 초기화

if (x < 2) { // 2 미만의 숫자는 소인수분해 불가

document.getElementById("result").innerHTML = "소인수분해에 실패하였습니다. " + x;

return;

}

do {

if (prime(i) == true) { // i 가 소수인지 확인

if (x % i == 0) { // i 가 x 의 인수인지 확인

while (x % i == 0) { // 인수로 나눌 수 있을 때까지 나누기

x /= i;

exp += 1; // 지수 증가

}

str += (i + "<sup>" + (exp != 1 ? exp : '') + "</sup>" + " × "); // 결과 문자열에 추가

exp = 0; // 지수 초기화

}

}

i += (i == 2) ? 1 : 2; // 2 이후로는 홀수만 체크

} while (x != 1); // x 가 1 이 될 때까지 반복

str = str.slice(0, -2); // 마지막의 "× " 제거

document.getElementById("result").innerHTML = str; // 결과 출력

}

이 함수는 factor() 함수로 호출되며, 소인수분해의 과정을 거쳐 #result 요소에 최종 결과를

출력합니다. 이 과정을 단계별로 살펴보겠습니다.

1. 변수 초기화

• x = document.getElementById("num").value: x 는 사용자가 입력한 숫자입니다.

• i = 2: 소인수분해를 시작할 가장 작은 소수(2)로 초기화합니다.

• exp = 0: 현재 인수의 지수(몇 번 나누었는지) 초기값을 설정합니다.

• str = x + " = ": 결과를 표현할 문자열을 준비합니다. 나중에 소인수들이 추가되며,

최종적으로 결과를 출력합니다.

2. 입력 값 확인 (if (x < 2))

• 설명: 소인수분해는 2 이상의 자연수에서만 의미가 있습니다. 따라서 2 미만의 숫자가

입력되면 소인수분해를 시도하지 않고 "소인수분해에 실패하였습니다."라는 메시지를

출력합니다.

3. 소수 및 인수분해 반복 (do...while 루프)

이 루프는 입력 숫자를 소인수로 나눌 수 없을 때까지 반복합니다.

• prime(i) == true: 현재 숫자 i 가 소수인지 확인합니다. 소수가 아닌 수는 소인수가 될 수

없으므로 소수가 아니면 건너뜁니다.

• if (x % i == 0): x 가 현재 소수 i 로 나누어 떨어질 때만 인수로 사용할 수 있습니다.

4. 소인수분해 및 지수 계산 (while (x % i == 0))

이 while 루프는 x 를 i 로 나눌 수 있을 때까지 나눕니다. i 로 나누어 떨어지지 않게 될 때까지

나누면서 exp 값을 증가시킵니다.

• 설명: x 를 계속 나누면, 특정 소수의 인수로 몇 번 나누어지는지를 나타내는 지수(exp)가

계산됩니다. 예를 들어, x 가 18 이라면 2^1 × 3^2 의 형태가 됩니다.

• str += (i + "<sup>" + (exp != 1 ? exp : '') + "</sup>" + " × ");: 나누어 떨어지지 않으면 현재

소수 i 와 지수 exp 를 결과 문자열에 추가합니다. exp 가 1 이면 sup 태그로 표시하지

않습니다.

5. 다음 소수로 이동 (i += (i == 2) ? 1 : 2)

• 설명: i 가 2 일 때만 1 을 더하여 다음 값인 3 으로 넘어가고, 그 이후로는 홀수만 더하여

소수를 확인합니다. (모든 짝수는 2 로 나누어 떨어지므로 홀수만 확인)

(삼항 연산자를 풀어서 작성) if (i == 2) {

i += 1; // i 가 2 인 경우에는 1 을 더해서 i 를 3 으로 만듭니다.

} else {

i += 2; // i 가 2 가 아닌 경우에는 2 를 더해서 다음 홀수로 만듭니다.

}

6. 결과 문자열 마무리 (str = str.slice(0, -2))

• 설명: 마지막에 남은 "× "를 제거하여 결과가 보기 좋게 완성됩니다.

7. 결과 출력 (document.getElementById("result").innerHTML = str;)

• 설명: 최종적으로 소인수분해된 결과 문자열을 HTML 요소 #result 에 출력하여 사용자가

결과를 볼 수 있게 합니다.

요약

이 함수는 입력된 숫자를 가장 작은 소수부터 나누어 나가며 소인수분해를 수행하고, 그 결과를

지수 형태로 표현하여 출력합니다.
